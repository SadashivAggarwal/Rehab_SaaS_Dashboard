# Rehab Outcomes SaaS Dashboard

A simulated SaaS web dashboard for physical therapy clinics, allowing users to upload, analyze, and download anonymized patient outcomes data. Built with Python (Streamlit), supports cloud deployment.

## Features
- Cloud-based demo (Streamlit Cloud-ready)
- Upload or use sample PT data
- Key metrics: FIM gain, visits, pain reduction
- Visual analytics (Plotly)
- Download reports (CSV)
- User roles: Therapist/Admin (simulated)
- All data is synthetic
